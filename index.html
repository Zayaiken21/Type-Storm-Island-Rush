How to Play

Type the active enemy prompt before it reaches your base. Tap an enemy to target it. Backspace edits. Enter confirms phrases.

Controls

Desktop: physical keyboard. Mobile: the built-in game keyboard only, so the phone keyboard stays hidden. Pause from the top-right HUD button.

Original Assets

All creatures are procedural Canvas drawings. Audio is generated with Web Audio. No external paid assets or copied branding are used.

Multiplayer
